C#/Unity

I spent a semester during my senior year of high school doing an independent study on C#, and subsequently applying it in Unity, a free game engine. The result of this work was a small (but mostly complete) 3D platformer based on an old game called Marble Blast, which I aptly named "Not Marble Blast." It was fun to not only learn a new programming language, but to also apply it to developing a game in a 3D engine, something that was new for me. Since I was mostly focused on the programming aspect of this, I did neglect other parts of the game, primarily art and sound. Nevertheless, I really enjoyed making this project, and am very happy with how it turned out. If I have time at some point in the future, I may revisit this in order to brush it up and perhaps expand it. For now, you can take a look at the source (I did my best to comment the code extensively), download a Windows build of the game, or simply watch a short demo of it below

Python

I took two general programming courses in high school, in which I learned the basics of programming in Python. Our final project for these classes was to create a game using PyGame, a game engine/library for Python. I decided to attempt to recreate Guitar Hero III within PyGame. I realized that this task was beyond the scope of what PyGame was probably meant to be used for, but I was mostly successful in my attempt. You find the source code, along with a write-up about the difficulties I faced while creating this, here and/or watch the demonstration video below.
